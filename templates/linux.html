    WELCOME TO LINUX!

    Linux is a Unix-like computer operating system assembled under the model of free and open-source software development and distribution. The defining component of Linux is the Linux kernel,an operating system kernel first released on September 17, 1991 by Linus Torvalds. The Free Software Foundation uses the name GNU/Linux to describe the operating system, which has led to some controversy. Linux was originally developed for personal computers based on the Intel x86 architecture, but has since been ported to more platforms than any other operating system. Because of the dominance of the Linux kernel-based Android OS on smartphones, Linux has the largest installed base of all general-purpose operating systems.

    Linux is also the leading operating system on servers and other big iron systems such as mainframe computers, and is used on 99.6% of the TOP500 supercomputers.It is used by around 2.3% of desktop computers.The Chromebook, which runs the Linux kernel-based Chrome OS, dominates the US K–12 education market and represents nearly 20% of the sub-$300 notebook sales in the US. Linux also runs on embedded systems – devices whose operating system is typically built into the firmware and is highly tailored to the system. This includes TiVo and similar DVR devices, network routers, facility automation controls, televisions, video game consoles and smartwatches.

    LINUX COMMANDS

    man = manual page.

    pwd = present working directory

    cd = change directory

    cd .. = takes you back to the previous directory

    cd ~ = takes you back to the route directory

    ls = list all files and folders in a directory

    ls -a = list all files and folders including the dot files

    LINUX EDITING COMMANDS

    pico = is an editing program,that is used for typing.It can be used to save files that can later be retreived.

    Ctrl O = this is used in saving a folder to pico

    Ctrl X = this is used in closing a file in pico

    cat"filename" = this is a command used to display a file

    head -1 "filename" = this displays the top text of a saved file

    tail -1 "filename" = this displays the bottom text of a saved file

    Note "filename" is a name of a file you have saved in pico

    ADVANCED COMMANDS

    ps aux = this is a command that prints running process
